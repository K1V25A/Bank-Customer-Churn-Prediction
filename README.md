# Bank Customer Churn Prediction

## Project Overview
This project was completed as part of the Slash Mark Business Analyst Internship. The objective is to predict customer churn using machine learning techniques to help banks identify customers at risk of leaving and develop effective retention strategies.

## Business Problem
Customer churn is a major challenge for banks, resulting in revenue loss and increased customer acquisition costs. This project predicts customer churn so that businesses can proactively retain valuable customers.

## Dataset
- **Source:** Kaggle – Telco Customer Churn Dataset
- **Records:** 10,000 customer records
- **Features:** 17
- **Target Variable:** Exited (0 = No Churn, 1 = Churn)

## Project Results
The models were evaluated using multiple performance metrics. Random Forest and XGBoost achieved the best overall performance after hyperparameter tuning, providing effective predictions for customer churn.
